기술 스택

// Engine & Languages

Unity 3D/2D

오브젝트 풀링(Object Pooling) 및 싱글톤 매니저 패턴을 적용하여 메모리(GC) 스파이크를 제어할 수 있습니다.

C#

주력 언어. 척력 기반 등의 물리/수학적 연산 로직을 구현하고 모듈 간 결합도를 낮추는 객체 지향적 코드를 작성합니다.

C++

기본적인 문법과 메모리 관리 구조(포인터, 참조)에 대한 기초적인 이해를 갖추고 있습니다.

// Backend & Database

Java & Spring

Java의 객체 지향 특성을 잘 이해하고 있으며, Spring 프레임워크를 활용해 기본적인 백엔드 API 서버를 구축하고 클라이언트와 연동해 본 경험이 있습니다.

DBMS & SQL

관계형 데이터베이스(RDBMS)의 설계 구조를 이해하고 있으며, SQL 쿼리를 직접 작성하여 데이터를 효율적으로 저장, 조회 및 관리할 수 있습니다.

// Network

Photon Fusion

Host 모드 기반의 멀티플레이어 환경에서 클라이언트 간의 상태 동기화 및 네트워크 로직을 구현할 수 있습니다.

// Architecture & Tools

AWS

캡스톤 프로젝트를 통해 EC2/RDS 기반의 서버 배포 경험이 있습니다.

Git & GitHub

Git을 이용한 형상 관리에 익숙하며 능동적인 사용이 가능합니다.
